# Assignment Creator Skill

This skill enables the automatic generation of comprehensive educational assignments with MCQs, rubrics, and structured tasks.

## Features

- **Objective Generation**: Creates clear learning objectives based on provided context
- **Task Description**: Develops detailed instructions and requirements for students
- **Grading Rubrics**: Designs comprehensive evaluation criteria with point values
- **MCQ Generation**: Produces 5 multiple-choice questions with 4 options each
- **Answer Keys**: Defines correct answers for all MCQs
- **Question Variety**: Mixes knowledge-based and application-based questions
- **Complete Structure**: Provides assessment title, task description, and rubrics

## How to Use

1. Provide the subject context or topic for the assignment
2. Optionally specify the target audience level
3. Receive a complete assignment package including:
   - Assessment title
   - Task description
   - Grading rubrics
   - 5 MCQs with four options each
   - Correct answers for all questions

## Output Components

1. **Assessment Title**: Descriptive title reflecting the assignment content
2. **Task Description**: Detailed instructions for students
3. **Grading Rubrics**: Clear evaluation criteria with point values
4. **Objective Questions**: 5 MCQs mixing knowledge and application-based questions
5. **Answer Keys**: Correct answers for all MCQs

## Question Types

- **Knowledge-Based**: Test recall and understanding of facts and concepts
- **Application-Based**: Require students to apply concepts to new situations

## Skill Components

- `SKILL.md`: Main skill definition and capabilities
- `references/creation_guide.md`: Detailed guidelines for assignment creation
- `scripts/sample_assignments.md`: Examples of complete assignments generated by the skill

## Best Practices

- Provide clear subject context for better assignment generation
- Specify target audience for appropriate difficulty level
- Review generated assignments for specific course alignment
- Customize rubrics based on institutional standards
