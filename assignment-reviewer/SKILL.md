# Assignment Reviewer Skill

This skill enables detailed review of programming assignments, including problem statement analysis, solution generation, and mistake identification in submissions.

## Features

- **Problem Statement Analysis**: Understand and break down assignment requirements
- **Solution Generation**: Provide correct solutions in code, pseudocode, or flowchart form
- **Mistake Identification**: Find logical, syntactic, and algorithmic errors in submissions
- **Flowchart Creation**: Generate visual representations of solutions
- **Constructive Feedback**: Offer actionable suggestions for improvement

## How to Use

1. Provide the problem statement to be reviewed
2. Include the student submission (code, pseudocode, or flowchart)
3. Optionally specify evaluation criteria
4. Receive detailed feedback with identified issues and suggestions

## Example Usage

```
"Please review this assignment submission for calculating Fibonacci numbers and identify any mistakes."
```

```
"Create a flowchart solution for sorting an array using bubble sort."
```

```
"Analyze this pseudocode for finding prime numbers and suggest improvements."
```
